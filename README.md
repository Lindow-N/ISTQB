# 🎯 ISTQB — Fiches de révision (Niveau Fondation v4.0)

Site statique de révision pour l'examen **ISTQB Certified Tester — Niveau Fondation v4.0**.
6 fiches mémo interactives + flashcards, reliées entre elles et classées par rentabilité.

## 🚀 Utilisation

- **En ligne (GitHub Pages)** : voir la section *About* du dépôt pour le lien.
- **En local** : ouvre `index.html` dans un navigateur.

## 🎓 L'examen en bref

- **40 questions** à choix unique · **60 minutes**
- **65 % de réussite** (26 bonnes réponses minimum)
- Pas de pénalité (les mauvaises réponses ne retirent pas de points)

Basé sur le syllabus officiel ISTQB v4.0 (français). Bonne révision ! 🍀
